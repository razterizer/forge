# Header-only library example

This example packages a header-only library without compiling a binary.

From this directory:

```sh
../../build/dev/forge build
../../build/dev/forge box create
../../build/dev/forge box list
../../build/dev/forge box inspect forge-example-header-only-0.1.0-ho.cbox
../../build/dev/forge box verify forge-example-header-only-0.1.0-ho.cbox
```

Bare box filenames are resolved from `.forge/boxes/`.
